# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-09

### Added
- Core MCP server implementation for JavaFX introspection.
- Tools: `ui_get_snapshot`, `ui_query`, `ui_perform`, `ui_get_node`, `ui_screenshot`.
- Java Agent support for non-intrusive attachment.
- Multi-module project structure.
- CI/CD workflows for GitHub Actions.
- Documentation: Manual, Tools Reference, and Specification.
